TESTE DE VIDA (HELLO WORLD)

O erro 500 persiste, o que indica que provavelmente o Python que estamos tentando usar não existe ou não tem as bibliotecas instaladas.

Vamos fazer um teste radical: rodar um "Hello World" usando o Python do sistema, sem nenhuma biblioteca.

PASSO 1: ATUALIZAR O ARQUIVO PYTHON
1. Pegue o arquivo `backend/passenger_wsgi.py` (que agora é bem pequeno).
2. Envie para `/apps_wsgi/checklist` no servidor (substitua o antigo).

PASSO 2: ATUALIZAR O .HTACCESS
1. Pegue o arquivo `backend/static/.htaccess`.
   - Note que mudei o Python para `/usr/bin/python3` (o padrão do sistema).
2. Envie para `/www/vrdchecklist.com.br` no servidor.

PASSO 3: TESTAR
1. Acesse https://checklist.vrdsolution.com.br/
2. Se aparecer "Hello World! Python is running":
   - SUCESSO! O servidor funciona.
   - O problema é que você não instalou as bibliotecas (FastAPI, etc) no servidor.
   - Precisaremos instalar via SSH ou enviar a pasta `venv`.

3. Se der erro 500 de novo:
   - O problema é mais grave (permissão de pasta ou configuração do KingHost).
