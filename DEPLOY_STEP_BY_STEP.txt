DEPLOYMENT STEPS FOR KINGHOST

1. PREPARE THE SERVER (FTP)
   - Navigate to `/www/vrdchecklist.com.br`
   - DELETE ALL FILES in this folder (index.php, default.html, etc).
   - This folder must be EMPTY except for the .htaccess file we will upload.

2. UPLOAD .HTACCESS
   - Locate the file `backend/static/.htaccess` on your computer.
   - Upload this file to `/www/vrdchecklist.com.br` on the server.
   - This file tells the server to load your Python application.

3. VERIFY APP FOLDER
   - Navigate to `/apps_wsgi/checklist` on the server.
   - Ensure `passenger_wsgi.py` exists in this folder.
   - Ensure the `app` folder exists inside.

4. TEST
   - Access https://checklist.vrdsolution.com.br/
   - You should see the Login page.
